Adding new Movie, Showtime and Price

The idea here is to add 1 Movie-Showtime-Price Object for each showing of a particular movie and push it into a Movie Array - for example:

  • Love Actually - 11am - 4.99
  • Love Actually - 2pm - 6.99
  • Love Actually - 5pm - 8.99
  • Love Actually - 8pm - 11.99
  • Notting Hill - 11am - 4.99
  • etc...

Preview Available Movies and Showtimes

  • The "load" button should load movies
  • The "pickMovie" button should have already taken the current movie information and populate fields for the next section
  • The "load" button should also enable the "next", "prev" and "pickMovie" buttons
  • The "next" and "prev" buttons will update the U/I with the appropriate Object information stored in that Array Element
  • The "pickMovie" button should take the currently displayed movie information and populate fields for the next section

Purchase Movie Tickets

  • The "pickMovie" button in the last section should have already taken the current movie information and populate fields for this section
  • The "pickMovie" button in the last section should also have already captured the Ticket Pricing information for you to use in your calculation
  • You can get the Number of Tickets from the UI and Calculate the total cost and display a user-friendly response as feedback to the user regarding their purchase by clicking on the "calculate Total" button
